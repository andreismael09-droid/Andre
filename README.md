#Generador de Contraseñas

##Descripción

Este proyecto consiste en un generador de contraseñas desarrollado en Python como parte de la asignatura Lógica de Programación.

El programa permite generar contraseñas aleatorias, crear contraseñas manualmente y evaluar su nivel de seguridad mediante estructuras de control y funciones.

##Funcionalidades
- Generar contraseñas aleatorias.
- Crear contraseñas manualmente.
- Evaluar la seguridad de una contraseña.
- Menú interactivo para acceder a las diferentes opciones.

##Estructuras utilizadas
Variables
Funciones
Condicionales (if, elif, else)
Bucles (while y for)
- Librería random
